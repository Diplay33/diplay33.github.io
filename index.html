Politique de confidentialité

Notre application mobile ne collecte aucune donnée personnelle de ses utilisateurs. Cependant, nous utilisons une API tiers qui peut collecter certaines informations lors de l'utilisation de notre application. Ces informations peuvent inclure votre adresse IP, votre emplacement géographique approximatif et des données sur votre utilisation de notre application. Nous utilisons ces informations uniquement dans le but de fournir une expérience utilisateur optimale et de maintenir la sécurité de notre application. Nous ne partageons pas ces informations avec des tiers ou des annonceurs. En utilisant notre application, vous acceptez que nous utilisions une API tiers qui peut collecter certaines informations vous concernant. Si vous avez des questions concernant notre politique de confidentialité ou l'utilisation des données collectées par l'API, n'hésitez pas à nous contacter.
